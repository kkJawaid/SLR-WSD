# Systematic Literature Review 

### Topic:
Knowledge-Augmented Urdu Reading System for Context-Sensitive Word Sense Disambiguation: Design, Fine-Tuning, and Integration of Lexical Resources

### Description:
This project proposes the design and development of a knowledge-augmented reading application specifically tailored to address context-sensitive word interpretation in Urdu. The system builds upon the Alif-1.0-8B-Instruct model through fine-tuning, leveraging structured lexical resources such as the Urdu WordNet Dataset and the ULS-WSD-18 Corpus. By integrating these resources, the system aims to enhance semantic understanding and provide precise, context-aware meanings to users in real time.
